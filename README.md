# IGG Technik

Willkommen im GitHub-Repository von IGG Technik!

## Erste Schritte

1.  **Dependencies installieren:**

    ```bash
    pnpm i
    ```

2.  **Backend einrichten:**

    ```bash
    pnpm dev:setup
    ```

3.  **Dev Server starten:**

    ```bash
    pnpm dev
    ```

Nach dem Start ist die Website unter [http://localhost:3001](http://localhost:3001) erreichbar.
